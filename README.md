# Payroll-Automation-Overtime-Calculation-System-Using-Excel

## Project Overview
The Payroll Automation & Overtime Calculation System was developed in Microsoft Excel to automate employee salary computation, overtime calculations, statutory deductions, and payroll summaries.
The objective was to eliminate manual payroll processing errors, improve efficiency, and provide management with accurate payroll reporting.
The solution supports multiple pay rates, overtime calculations, automated deductions, and payroll summaries.

## Business Problem

Many small and medium-sized organizations process payroll manually, leading to:

- Payroll calculation errors
- Inconsistent overtime calculations
- Delayed salary processing
- Lack of payroll transparency
- Difficulty tracking payroll costs

The goal was to develop an automated payroll solution capable of:

- Handling different employee pay rates
- Calculating overtime automatically
- Applying deductions consistently
- Generating payroll summaries for management

## Tools Used
Microsoft Excel

Used for:
- Data Entry
- Formula Calculations
- Payroll Reporting
- Payroll Dashboard
